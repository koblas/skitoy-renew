Ended up wondering what it would take to create the blink tag in jQuery. Â It was pretty easy, but the big catch is that $(this).is(&#8220;:visible&#8221;) is really checking &#8220;display == none&#8221;.

That said, here&#8217;s the code.

```html
<p>test <span class="blink">HIHIHI</span> of blink</p>
<p>test <span class="blink">HIHIHI</span> of blink</p>

<script>
setInterval(function() {
   $('.blink').each(function() {
     $(this).css('visibility',
         $(this).css("visibility") == 'hidden' ? '' : 'hidden'); });
   }, 500);
</script>
```
